# 👋 ¡Hola! Soy Manuel

## 💻 Sobre mí

Soy desarrollador especializado en **plugins para servidores de Minecraft**, creando soluciones personalizadas, optimizadas y escalables para comunidades de todo tipo.

Cuento con **varios años de experiencia** desarrollando plugins, sistemas y herramientas para servidores, enfocándome en ofrecer código limpio, eficiente y fácil de mantener.

Siempre estoy buscando mejorar mis conocimientos y aprender nuevas tecnologías para crear proyectos cada vez más completos.
